Como obter a URL:
1. Abra a planilha Google Sheets → Extensões → Apps Script
2. Cole o código do arquivo Code.gs entregue junto
3. Clique em Implantar → Nova implantação
4. Tipo: Aplicativo web · Executar como: Eu · Acesso: Qualquer pessoa
5. Copie a URL gerada e cole aqui acima
